git remote - чтобы просмотреть, какие удалённые серверы у вас уже настроены
git remote -v - Чтобы посмотреть, какому URL соответствует сокращённое имя в Git
git clone [url] - клонирование репозитория 
git remote add [сокращение] [url] - Чтобы добавить новый удалённый Git-репозиторий под именем-сокращением, к которому будет проще обращаться

git fetch pb - извлечь всю информацию, которая есть в репозитории bp

Сохранение состояния
$ git init
$ git add .
$ git commit -m "Моя первая резервная копия"


$ git reset --hard - восстановить первоначальную версию

git log - список коммитов
git log -p - выводит изменения, которые были сделаны после предыдущего коммита
